Java 重要名詞

  • write once, run anywhere
  1. JDK : Java Development Kit 開發工具包
    1. JDK裡包含了JRE，也包含JVM，函式庫，所以下載JDK即可
  2. JVM : Java Virtual machine 虛擬機器
    1. 是Java能夠跨平台的關鍵(write once,run anywhere)
  3. IDE : Integrated Development Environment 開發整合環境
    1. Ex:eclipse IntelliJ IDEA
  4. SDK : Software Development Kit 軟體開發工具
    1. JDK就是一個SDK是Oracle提供給我們的一個Java的SDK

Java 基本資料型態 Primitive Type &初始值

  • 又可區分 : 整數，浮點數，字元，布林型
  • 資料型態 小 -> 大:
  • byte → short → char → int → long → float → double
// 1. 資料轉換 小轉大：
byte a = 100;
int b = a; //這也可以 int b = (int)a;

// 2. 資料轉換 大轉小 （強制轉換）
int c = 100;
byte d =(byte) c;

// 3. 整數轉字串:
int e = 100; 
String f = Integer.toString(e);

// 4. 字串轉整數:
String i = "100";
int j = Integer.parseInt(i);

Java 編譯過程

運算子功能介紹

  • 運算子執行運算之後將回傳值，而其回傳值型態是運算元而定
  • 以下運算子先後順序:

算數運算

遞增遞減運算子

  • 後置型(a++)先取值再遞增，前置型(++a)先遞增再取值

指定運算子

  • 等號右值的位階不可高於左邊的位階
  • 位階高低順序:double > float > long > int > short > byte

關係運算子(Relational operators)

  • 關係運算子:用來比較兩個變數的值，其回傳結果是一個布林值

條件運算子(Conditional operators)

  • 是將兩個布林值合併起來的運算子，運算結果仍為布林值
  • 請使用2個 & | 進行合併，因為Java運算時會以短路運算的邏輯進行

三元運算子(Ternary operators)

  • 是一個簡略的if-else敘述式

位元運算子(Bit operators)

  • &(and) | (or) ^(xor)互斥 可用在整數的位元
  • ~(位元反轉運算子):用於整數 將位元1變成0，0變成1
